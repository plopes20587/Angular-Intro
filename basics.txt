
1. A user visits the AngularJS app.

2. The view presents the app's data through the use of expressions, filters, and directives.

3. Directives bind new behavior HTML elements.

4. A user clicks an element in the view. If the element has a directive, AngularJS runs the function.

5. The function in the controller updates the state of the data.

6. The view automatically changes and displays the updated data. The page doesn't need to reload at any point.
